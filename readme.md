========================================================================

\[안내] 도면목록표 ↔ 개별 도면 자동 검토 프로그램\_CAD (v\_0.1)

========================================================================

\[버전]

260415 v\_0.1 : 초기 모델



1\. 개요

&#x20;  본 프로그램은 도서 납품 전, '도면목록표'와 '개별 캐드 도면'의

&#x20;  타이틀 정보를 자동으로 교차 검토하여 데이터 불일치를 잡아내는 도구입니다.



2\. 주요 기능

&#x20;  - 도면목록표(DWG) 내 도면번호, 도면명, 축척(A1/A3) 자동 추출

&#x20;  - 수백 개의 개별 도면(DWG) 타이틀 박스 정보 일괄 추출 (멀티코어 병렬 처리)

&#x20;  - 목록표 vs 실제 도면 불일치 항목 및 누락 도면 엑셀 리포트 생성 (빨간색 강조)



#### 3\. **필수 사전 설치 사항 (중요!)**

&#x20;  본 프로그램은 캐드 파일 읽기를 위해 외부 엔진인 'ODA File Converter'가 필요합니다.

&#x20;

&#x20;  - 다운로드 링크: https://www.opendesign.com/guestfiles/oda\_file\_converter

&#x20;  - 설치 방법: 위 링크에서 .msi 파일을 받아 설치하세요.

&#x20;  - \[주의] 설치 시 경로를 변경하지 마세요. (기본값: C:\\Program Files\\ODA)

&#x20;    만약 경로를 변경할 경우 프로그램이 엔진을 찾지 못해 실행되지 않습니다.



4\. 사용 방법

&#x20;  ① 프로그램(.exe) 실행

&#x20;  ② 안내에 따라 '도면목록표 DWG' 파일 경로 입력 (마우스로 끌어다 넣으셔도 됩니다)

&#x20;  ③ 개별 도면들이 모여 있는 '폴더 경로' 입력 (하위 폴더는 스캔하지 않으므로 주의)

&#x20;  ④ 도곽 블록 이름 입력 (예: XR-form)

&#x20;  ⑤ 완료 후 생성된 '도면검토리포트.xlsx' 확인



5\. 주의 사항

&#x20;  - 도곽 블록 이름이 정확해야 정보를 가져올 수 있습니다.

&#x20;  - 검토하려는 도면 파일이 다른 프로그램(AutoCAD 등)에서 열려 있어도 작동합니다.

&#x20;  - 분석 결과가 'X'로 표시되거나 빨간색으로 뜬 항목은 수동 확인이 필요합니다.



6\. 배포 및 문의

&#x20;  - 제작/배포: \[건원건축 설계사업 6본부 1소 김정현]

&#x20;  - 본 프로그램은 팀 내 업무 효율화를 위해 제작되었으며, 외부 유출을 금합니다.
