[안내] 도면목록표 ↔ 개별 도면 자동 검토 프로그램 (AutoDWG Cross-Checker V1.4)
========================================================================

[버전 업데이트]
V1.4 : LISP 연동(사용자 지정 구역), 스마트 매칭, 스케일 자동 인식, 세로 도곽(회전) 감지 탑재

1. 개요
  본 프로그램은 도서 납품 전, '도면목록표'와 수백 장의 '개별 캐드 도면' 타이틀 정보를 
  자동으로 교차 검토하여 데이터 누락 및 불일치를 찾아내는 업무 자동화 솔루션입니다.


2. 주요 기능
  - [NEW] 사용자 맞춤형 구역 설정: LISP을 이용해 마우스 드래그만으로 도면명, 번호, 축척 구역을 직관적으로 지정 (글자가 없는 빈 표 등 모든 형태의 도곽 완벽 대응).
  - [NEW] 스마트 스케일 & 회전 감지: 1:1, 1:100 등 도곽 스케일에 상관없이 원본 비율을 자동 계산하며, 세로로 90도 회전된 도곽도 정확히 역산하여 스캔합니다.
  - [NEW] 스마트 매칭 로직: 도면번호 비교 시 하이픈(-), 언더바(_), 띄어쓰기를 무시하여 완벽하게 매칭합니다.
  - 멀티코어 병렬 처리로 수백 장의 도면을 단 몇 초 만에 고속 스캔.
  - 불일치 항목 및 누락 도면 엑셀 리포트(빨간색 셀 강조) 자동 생성.


3. 필수 사전 설치 사항 (중요!)
  이 프로그램이 백그라운드에서 캐드 파일을 고속으로 읽어오려면 'ODA File Converter'라는 분석 엔진이 반드시 필요합니다.

  - 다운로드 링크: https://www.opendesign.com/guestfiles/oda_file_converter
  - 설치 방법: 위 링크에서 .msi 파일을 받아 설치하세요.
  - [주의] 설치 시 경로를 변경하지 마세요. (기본값: C:\Program Files\ODA)
    경로를 임의로 변경할 경우 프로그램이 엔진을 찾지 못해 실행되지 않습니다.


4. 사용 방법 (핵심)
  프로그램은 [캐드에서 구역 설정하기]와 [검토 프로그램 실행하기] 두 단계로 나뉩니다.

  [1단계] 오토캐드에서 도곽 구역 설정하기 (프로젝트당 최초 1회만 실행)
  ① 오토캐드를 켜고 검토할 도면 중 아무거나 한 장을 엽니다. (도면 스케일 무관)
  ② `APPLOAD` 명령어를 입력하여 동봉된 `SET_ROI.lsp` 파일을 로드합니다.
  ③ 명령어 창에 `SET_ROI`를 입력하고 실행합니다.
  ④ 안내에 따라 기준이 될 '도곽 블록'의 외곽선을 클릭하여 선택합니다.
  ⑤ 마우스를 드래그하여 [도면명], [도면번호], [축척] 구역(박스)을 순서대로 지정합니다.
  ⑥ 설정이 완료되면 윈도우 백그라운드에 구역 좌표가 자동 저장됩니다. 이제 캐드를 닫아도 됩니다.

  [2단계] 검토 프로그램 실행하기 (도면 수정 시 수시로 실행)
  ① `도면자동검토기.exe` (또는 app.py)를 실행합니다.
  ② 1단계에서 클릭했던 도곽 블록의 이름(예: XR-form)을 정확히 입력합니다. 
     (※ 입력이 틀리거나 저장된 파일이 없으면, 현재 저장된 도곽 힌트 목록을 보여줍니다.)
  ③ [도면목록표 DWG 파일]의 전체 경로를 복사하여 붙여넣습니다.
  ④ 검토할 개별 도면들이 모여 있는 [폴더 경로]를 입력합니다. 
     (여러 폴더 입력 가능, 다 넣었으면 'N' 입력)
  ⑤ 잠시 기다리면 터보 모드로 검토가 완료되며, 프로그램이 있는 폴더에 `도면검토리포트_최종.xlsx` 파일이 생성됩니다!


5. 주의 사항
  - 도곽은 반드시 EXPLODE(분해) 되지 않은 '블록(Block)' 상태여야 LISP이 전체 크기와 회전 각도를 정상적으로 인식할 수 있습니다.
  - 파이썬에서 입력하는 도곽 블록 이름은 대소문자, 띄어쓰기가 캐드 블록명과 완벽히 일치해야 합니다.
  - 검토하려는 도면 파일이 다른 프로그램(AutoCAD 등)에서 열려 있어도 검토는 정상 작동합니다. (단, 엑셀 리포트 파일이 켜져 있는 상태에서 프로그램을 다시 돌리면 덮어쓰기 에러가 발생하므로 엑셀 창은 닫고 실행해 주세요.)
  - 분석 결과 엑셀에서 빨간색으로 강조된 항목은 반드시 수동으로 원본 도면을 확인하시기 바랍니다.
